# BCGov Form.io components library

This module contains BCgov components for use with Form.io. Any code within this library should be considered examples, which may be outdated. While these serve as an example for how components have been extended, it is not intended for use within a runtime application. 

## Installation
To install this library, do the following.

```
npm install --save bcgov/contrib
```

## Usage

```javascript
import { Formio } from 'formiojs';
import BCGovFormioComponents from 'bcgov/contrib';
Formio.use(BCGovFormioComponents);
```
